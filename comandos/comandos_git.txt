
git init . 		-> Torna o diretorio atual um repositorio git.
git status 		-> Mostra como está o diretorio.
git add 		-> ESCOLHE o que vai para o commit, mas NÃO é o commit.
git commit -m "primeiro comit bash" -> Esse comando que COMMITA.
git log 		-> Mostra um HISTORICO de todos os commits.



1 -> cria um repositório.
2 -> git init . -> inicia um repo git no diretório atual.
3 -> git add -> adiciona um arquivo para ser comitado ( ñ é o commit ).
4 -> git commit -m "mensagem estranha" -> commita aquilo que foi adicionado para ser commitado com o git add.
5 -> git status
6 -> volta para o passo 3
